```
Film
GET       /api/film

PUT       /api/film/{id}
DELETE    /api/film/{id}

Cinema
GET       /api/cinema
POST      /api/cinema
GET       /api/cinema/{id}
PUT       /api/cinema/{id}
DELETE    /api/cinema/{id}
```
